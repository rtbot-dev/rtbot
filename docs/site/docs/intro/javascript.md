# JavaScript


## Installation

`RtBot` can be run in the browser as well as in `NodeJs` and comes with `TypeScript` support.

In your project, just add `@rtbot/api` as any dependency:

```bash
# with npm
npm install @rtbot/api

# with yarn
yarn add @rtbot/api

# with pnpm
pnpm add @rtbot/api
```
